# 🧮 Tasbeeh Counter App (Advanced)

A simple and interactive **Tasbeeh Counter Web App** built using HTML, CSS, and JavaScript. This app allows users to count tasbeeh digitally with additional features like sound feedback and dynamic image changes.

---

## 🚀 Features

* ➕ **Increment Counter** – Increase count on each button click
* 🔄 **Reset Option** – Reset the counter back to zero
* 🔊 **Sound Feedback** – Plays a sound on every click
* 🖼️ **Dynamic Image Change** – Image updates based on count ranges
* 🎯 **Range Control** – Counter is limited within defined bounds

---

## 🧠 Concepts Used

* DOM Manipulation
* Event Handling (Click Events)
* Conditional Logic (if/else)
* State Management (count tracking)
* Media Handling (audio & images)

---

## 📂 Project Structure

* `HTML` → Structure of the app
* `CSS` → Styling and layout
* `JavaScript` → Logic and interactivity
* `Assets` → Images and audio files

---

## 🎯 How It Works

1. User clicks the **Add button**
2. Counter increases by 1
3. Sound plays instantly
4. Image changes based on predefined count ranges
5. Reset button restores everything to the initial state
